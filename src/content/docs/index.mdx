## Quick Start

1. **Register your app** — Create an account and [register your application](/apps/register/) to get an API key.
2. **Create a community** — Use the API to [create a community](/communities/create/) with a handle and profile.
3. **Add members** — Let users [join communities](/communities/members/) and manage membership.
4. **Store data** — Create and manage [ATProto records](/communities/records/) in community repositories.

Ready to dive in? Check out the [full API reference](/reference/api-endpoints/).
